# npm 패키지 등록 방법

1. `npm init`
   1. package name 작성(ex) @jsnooks/use-title)
   2. version 작성 (ex) 1.0.0)
   3. license: MIT 등 설정하기
2. `npm login`으로 로그인 하기
3. `npm publish --access public`으로 npm에 등록하기

## yarn

> `yarn add @jsnooks/use-title`

## npm
> `npm i @nooks/use-title`

해당 명령어로 패키지 설치 후 사용
